June Highlights

Thursday, June 4, 2015

NOTE: The chart has been updated this month to measure percentage of posts, smoothing out the change in overall number of posts as whoishiring becomes more popular.

Are you a Node.js fan? Then this is your lucky month. Node surpassed Rails as the #1 web application framework. After getting close and tying a couple of times last year, Node fell back a bit, tune-in next month to see if Node maintains momentum.

If you didn't know it was a JavaScript world in web development yet, allow me to demonstrate. Even though Python claimed the top spot, JS had both JS itself and AngularJS in the top 5. At the same time, React continued its blistening growth rate giving JS 4 entries in the top 20 (JS itself, AngularJS, Node.js and React).
