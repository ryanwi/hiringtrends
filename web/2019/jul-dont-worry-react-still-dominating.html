July Highlights

Sunday, July 28, 2019

Welcome back! Hacker News Hiring Trends returns from a Spring Break to get caught up on 2019 technology trends. It may come as a surprise, but React is still on top, running its win streak past 2 years. React first won in June 2017 and has been on top ever since. No other technology has enjoyed such a string of success in the 8 year history of "Who is hiring".

Since it has been a few months since I last posted, it does allow other areas of software development to experience a shake-up and surprisingly programming languages are seeing some interesting trends. Normally, programming languages have been fairly static relative to the faster moving area of frameworks, but a number of programming languages are appearing frequently in job postings. The Go programming language is establishing itself as the third most popular programming language and consistently placing in the Top 10 overall. Rust is another programming language making news as it records its first appearance in the Top 10 programming languages, knocking out C in the process.
