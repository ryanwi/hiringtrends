November Highlights

Thursday, November 13, 2014

JavaScript racks up its third win in a row with another strong showing, leaving Python to come in second again. Rails, Ruby and iOS round out the top 5. Elsewhere in the top 10, a little shakeup had node.js and AWS (amazon web services, i.e. "the cloud") coming up and Java taking a big tumble out of the Top 10.
